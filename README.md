# Titanic-EDA-Analysis
Performed exploratory data analysis on the Titanic dataset to identify survival patterns. Cleaned missing data and visualized insights using Python, Pandas, Matplotlib, and Seaborn. Found that gender, passenger class, and age significantly influenced survival rates, with females and first-class passengers having higher chances of survival.
##  Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns based on passenger attributes.


# Objectives
- Analyze survival patterns
- Handle missing data
- Visualize key insights


# Data Cleaning
- Filled missing Age values using median
- Filled missing Embarked values using mode
- Dropped Cabin column due to excessive missing values


# Key Questions Answered

### 1. Who survived more?
 Females had a significantly higher survival rate than males.

### 2. Did passenger class affect survival?
 Yes, passengers in higher classes had better survival chances.

### 3. Survival by age group?
 Children had higher survival rates compared to adults and seniors.

- Survival Rate by Passenger Class
- Age Distribution Histogram


##  Tools Used
- Python

## Conclusion
Gender, passenger class, and age were key factors affecting survival on the Titanic.
